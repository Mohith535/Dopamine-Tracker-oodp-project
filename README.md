# Digital Habit & Dopamine Tracker

A calendar-based behavioral tracking system designed to help users monitor daily habits, visualize consistency, and track dopamine scores. This project demonstrates core Object-Oriented Programming (OOP) concepts in C++ and includes a premium, responsive web frontend.

## 🌟 Features
- **Interactive Calendar**: Click any date to log your habits.
- **Dopamine Scoring**: Positive habits grant points (+10), visualizing your discipline.
- **Consistency Analytics**: Progress bars and streak counters for each individual habit.
- **Monthly Graph**: A dynamic line chart visualizing your performance over the month.
- **Local Storage**: Your data is saved directly in your browser.

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3 (Custom Design System), Vanilla JavaScript (Canvas API)
- **Backend Model**: C++17 (Demonstrating Abstraction, Inheritance, Polymorphism, and Encapsulation)

## 💻 Running Locally
### Frontend (UI Simulation)
Simply open the `index.html` file in any modern web browser. No server is required.

### Backend (C++ Logic Engine)
Compile and run the `main.cpp` file:
```bash
g++ -o main.exe main.cpp
./main.exe
```
