# 🧠 Digital Habit & Dopamine Tracker System

A simple yet structured behavioral tracking system built using **C++ (OOP concepts)** and a **calendar-based UI approach** to model human discipline, consistency, and productivity.

---

## 🎓 Academic Context

This project was developed as part of my **First Year – Second Semester** coursework for
**Object Oriented Design and Programming (OODP)**
at **SRM Institute of Science and Technology**.

---

## 💡 Project Intent

This system is designed to address a common modern problem:
lack of consistency, distraction, and unstructured daily habits.

It works by:

* Tracking daily habits
* Classifying them into **positive and negative behaviors**
* Converting actions into a **dopamine-based score**
* Visualizing **progress, consistency, and discipline over time**

## 📊 Key Features

- **Interactive Calendar**: Click any date to log your habits (Good / Bad).
- **Dopamine Scoring**: Positive habits grant points (+10), visualizing your discipline.
- **Consistency Analytics**: Progress bars and streak counters for each individual habit.
- **Monthly Graph**: A dynamic line chart and ASCII/graph-based visualization of your performance.
- **Local Storage**: Your data is saved directly in your browser for persistence.

## 💻 Running Locally

### Frontend (UI Simulation)
Simply open the `index.html` file in any modern web browser. No server is required.

### Backend (C++ Logic Engine)
Compile and run the `main.cpp` file:
```bash
g++ -o main.exe main.cpp
./main.exe
```
